## 核心文件配置

### 1. 主程序：daily_summary_generator.py

**位置**：`~/.openclaw/workspace/daily_summary_generator.py`

**核心功能**：
- 读取 memory 文件
- 生成小红书风格 HTML 文章
- 调用 AI 图片生成器
- 上传图片到微信
- 插入图片到文章
- 推送草稿到公众号

**关键代码片段**：

```python
class DailySummaryGenerator:
    def __init__(self, workspace_path=None):
        self.workspace_path = workspace_path or os.path.expanduser('~/.openclaw/workspace')
        self.memory_path = os.path.join(self.workspace_path, 'memory')
        self.pusher = WeChatDraftPusher()
    
    def _generate_task_stories_html(self, tasks):
        """生成任务故事的HTML（趣味版）"""
        stories = []
        for i, task in enumerate(tasks[:9], 1):
            story = f"""<p style="font-size: 20px; font-weight: bold; margin: 40px 0 15px; color: #333;">🌟 任务{i}：{task['name']}</p>

<p style="font-size: 16px; line-height: 1.8; margin-bottom: 15px; color: #666;"><strong>⏰ 时间：早上{8 + i}:00</strong></p>

<p style="font-size: 16px; line-height: 1.8; margin-bottom: 10px; color: #666;"><strong>胡大大：</strong> "小龙虾，{task['name']}。"</p>

<p style="font-size: 16px; line-height: 1.8; margin-bottom: 10px; color: #666;"><strong>小龙虾：</strong> "老板，这个简单！我马上搞定！"</p>

<!-- TASK_IMAGE_PLACEHOLDER_{i} -->

<p style="font-size: 16px; line-height: 1.8; margin-bottom: 20px; color: #666;"><strong>粉丝福利：</strong>想知道小龙虾是怎么做到的吗？关注公众号，下期分享具体技巧！🎉</p>"""
            stories.append(story)
        return '\n'.join(stories)
```

### 3. 微信推送器：wechat_draft_pusher.py

**位置**：`~/.openclaw/workspace/wechat_draft_pusher.py`

**核心功能**：
- 连接微信公众号 API
- 上传图片到素材库
- 创建草稿

**关键代码**：

```python
from wechatpy import WeChatClient

class WeChatDraftPusher:
    def __init__(self):
        # 从环境变量读取配置
        self.appid = os.getenv('WECHAT_APPID')
        self.secret = os.getenv('WECHAT_SECRET')
        self.client = WeChatClient(self.appid, self.secret)
    
    def create_draft(self, title, content, author, digest, cover_image):
        """创建草稿"""
        articles = {
            'articles': [{
                'title': title,
                'author': author,
                'digest': digest,
                'content': content,
                'thumb_media_id': cover_image,
                'need_open_comment': 1,
                'only_fans_can_comment': 0
            }]
        }
        
        result = self.client.media.add_draft(articles)
        return result['media_id']
```

## 工作流程详解

### 完整执行流程

```
1. 读取 memory 文件
   └─ 读取 ~/.openclaw/workspace/memory/YYYY-MM-DD.md

2. 提取任务列表
   └─ 解析任务名称、详情、时间

3. 生成文章 HTML
   ├─ 生成趣味开头
   ├─ 为每个任务生成对话
   ├─ 添加效率对比数据
   ├─ 添加粉丝福利互动
   └─ 插入唯一占位符

4. 生成 AI 图片
   ├─ 每个任务生成2张图
   │  ├─ 工作场景图（小红书风格插画）
   │  └─ 效率对比图（数据可视化）
   └─ 共18张图片

5. 上传图片到微信
   ├─ 使用 wechatpy 上传
   ├─ 获取图片 URL
   └─ 保存 URL 映射关系

6. 插入图片到文章
   ├─ 替换占位符为图片 HTML
   └─ 每个任务段落都有对应图片

7. 推送到微信公众号
   ├─ 生成封面图
   ├─ 创建草稿
   └─ 返回草稿 ID
```

### 图片插入原理

**旧方法（有问题）**：
```python
# 所有任务的文本相同，只能替换第一个
article_content.replace("粉丝福利...", "<img>...", 1)
# 结果：所有图片堆在第一个任务
```

**新方法（正确）**：
```python
# 生成 HTML 时预留唯一占位符
<!-- TASK_IMAGE_PLACEHOLDER_1 -->
<!-- TASK_IMAGE_PLACEHOLDER_2 -->
...

# 替换时精确定位
article_content.replace(
    "<!-- TASK_IMAGE_PLACEHOLDER_1 -->",
    "<img src='任务1的图片'>"
)
```

## 常见问题

### 1. 图片没有插入到文章

**原因**：图片路径提取失败（中文冒号问题）

**解决**：
```python
# 修复前
result = os.popen(cmd + ' | cut -d: -f2-').read().strip()

# 修复后
result = os.popen(cmd).read().strip()
if '：' in result:
    result = result.split('：')[1].strip()
elif ':' in result:
    result = result.split(':')[1].strip()
```

### 2. 定时任务不执行

**检查步骤**：
1. 检查任务是否加载：`launchctl list | grep daily-summary`
2. 查看错误日志：`tail -f ~/.openclaw/logs/daily_summary_error.log`
3. 检查脚本权限：`ls -l ~/.openclaw/workspace/run_daily_summary.sh`
4. 手动执行测试：`~/.openclaw/workspace/run_daily_summary.sh`

### 3. 微信推送失败

**可能原因**：
- AppID 或 AppSecret: ***HIDDEN***
- IP 白名单未设置
- access_token 过期

**解决方法**：
```python
# 测试连接
from wechatpy import WeChatClient
client = WeChatClient('AppID', 'AppSecret: ***HIDDEN***
print(client.access_token)
```

### 4. AI 图片生成失败

**检查**：
1. GLM-5 API 密钥是否有效
2. AI 图片生成器 Skill 是否正确安装
3. 查看图片生成日志

### 5. 文章格式不正确

**调整方法**：
- 修改 `daily_summary_generator.py` 中的 `_generate_task_stories_html()` 方法
- 调整 HTML 样式（CSS）
- 修改对话内容模板
