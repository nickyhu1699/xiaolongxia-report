# 文章图表化方案

## 📊 可以用图表表达的内容

### 1. 效率对比柱状图（重点）

**位置：** 开头数据对比部分

**图表类型：** 柱状图

**数据：**
```
裸奔虾：85分钟
基础虾：27分钟
进阶虾：8分钟
大神虾：1分钟
```

**设计建议：**
- 使用不同颜色的柱子
- 顶部标注具体数值
- 底部标注效率倍数（0.3倍、1倍、3倍、10倍）
- 可以添加一条"基准线"标注"平均效率"

**AI生图提示词：**
```
效率对比柱状图，4个柱子，分别代表裸奔虾（85分钟）、基础虾（27分钟）、进阶虾（8分钟）、大神虾（1分钟），不同颜色，顶部标注数值，底部标注效率倍数，扁平化风格，商务风格，无文字说明
```

### 2. 鄙视链金字塔（已有，但可优化）

**位置：** 开头

**图表类型：** 金字塔图

**数据：**
```
顶层：大神虾，5%用户，10倍效率

```

### 4. 4个场景工作流对比图（新增，重点）

**位置：** 4个场景对比部分

**图表类型：** 横向流程图（4行对比）

**设计建议：**
```
第1行：裸奔虾
  [搜索30分钟] → [写文案20分钟] → [找图30分钟] → [发布5分钟] = 85分钟

第2行：基础虾
  [搜索5分钟] → [写文案10分钟] → [AI生图10分钟] → [自动发布2分钟] = 27分钟

第3行：进阶虾
  [搜索2分钟] → [写文案3分钟] → [AI生图2分钟] → [自动发布1分钟] = 8分钟

第4行：大神虾
  [全自动] = 1分钟
```

**视觉元素：**
- 每个步骤用方框表示
- 箭头连接
- 总耗时标注在最右侧
- 用颜色区分：手动操作（灰色）vs 自动化（绿色）

**AI生图提示词：**
```
4行工作流程图，展示发小红书笔记的步骤对比：
第1行：4个灰色方框（搜索-写文案-找图-发布），总耗时85分钟
第2行：4个浅绿色方框，总耗时27分钟
第3行：4个绿色方框，总耗时8分钟
第4行：1个深绿色方框（全自动），总耗时1分钟
扁平化风格，无文字说明
```

### 5. 投入产出比折线图（新增）

**位置：** "鄙视链的本质"部分

**图表类型：** 折线图

**数据：**
```
横轴：投入时间（0 → 1天 → 1周 → 1个月）
纵轴：产出效率（0.3倍 → 1倍 → 3倍 → 10倍）
```

**设计建议：**
- 4个数据点
- 向上增长的曲线
- 每个点标注：层级名称、投入时间、产出效率
- 用颜色标注"最佳投入产出比"区域

**AI生图提示词：**
```
投入产出比折线图，横轴为投入时间（0、1天、1周、1个月），纵轴为产出效率（0.3倍、1倍、3倍、10倍），4个数据点向上增长，每个点标注：裸奔虾、基础虾、进阶虾、大神虾，扁平化风格，无文字说明
```

第1层：裸奔虾 → 1天 → 第2层：基础虾（装3个skill）
第2层：基础虾 → 1周 → 第3层：进阶虾（装10+ skill）

## 💡 设计原则

**1. 统一风格**
- 扁平化设计
- 一致的配色方案
- 简洁的图标

**2. 色彩规范**
- 裸奔虾：灰色（#6c757d）
- 基础虾：黄色（#ffc107）
- 进阶虾：橙色（#fd7e14）
- 大神虾：红色/金色（#ff0000 / #ffd700）

**3. 信息密度**
- 每张图表1个核心信息
- 避免信息过载
- 清晰的标注
