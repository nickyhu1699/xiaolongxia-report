## 🚀 快速使用指南

### **单篇笔记创作**

```bash
# 步骤1：搜集素材
你：@张骞 搜集"咖啡店装修"的20篇热门笔记

# 步骤2：规划选题
你：@诸葛亮 规划本周5篇选题

# 步骤3：创作文案
你：@李白 写第1篇"咖啡店吧台设计的5个避坑指南"

# 步骤4：制作配图
你：@鲁班 制作封面图和4张配图

# 步骤5：审核内容
你：@包拯 审核这篇笔记

# 步骤6：发布上线
你：确认发布
```

---

### **批量生产（并行模式）**

```bash
# 并行执行3个任务

# 小龙虾自动调度
我：[启动3个agent并行]
    张骞：素材搜集完成！

    鲁班：第2篇配图完成！
    汇总：3项任务并行完成，节省12分钟！
```
