# 飞书文档操作工作流

## ⚠️ 重要规则

### 创建带内容的文档（正确做法）

**❌ 错误做法：**
```javascript
feishu_doc({
  action: "create",
  title: "文档标题",
  content: "文档内容"  // ❌ 这个参数不会被处理！
})
```

**✅ 正确做法（两步操作）：**
```javascript
// 第一步：创建空文档
const result = await feishu_doc({
  action: "create",
  title: "文档标题"
})

const docToken = result.document_id

// 第二步：写入内容
await feishu_doc({
  action: "write",  // 或 "append"
  doc_token: docToken,
  content: "文档内容"
})
```

---

## 📋 飞书文档操作类型

### 1. create - 创建文档
- **只能创建空文档**
- **参数**：title, folder_token（可选）
- **返回**：document_id, url
- **❌ 不支持**：content 参数会被忽略

### 2. write - 写入内容（覆盖）
- **完全覆盖文档内容**
- **参数**：doc_token, content
- **用途**：首次写入或完全替换内容

### 3. append - 追加内容
- **在文档末尾追加内容**
- **参数**：doc_token, content
- **用途**：在现有内容后添加新内容

### 4. insert - 插入内容
- **在指定位置插入内容**
- **参数**：doc_token, content, after_block_id（可选）

### 5. read - 读取内容
- **读取文档内容**
- **参数**：doc_token

---

## 🎯 标准工作流程

### 创建新文档（带内容）
```javascript
// 1. 创建空文档
const createResult = await feishu_doc({
  action: "create",
  title: "文档标题",
  folder_token: "可选的文件夹token"
})

// 2. 写入内容
await feishu_doc({
  action: "write",
  doc_token: createResult.document_id,
  content: "完整的文档内容"
})

// 3. 返回结果
return {
  document_id: createResult.document_id,
  url: createResult.url
}
```

### 追加内容到现有文档
```javascript
await feishu_doc({
  action: "append",
  doc_token: "existing_doc_token",
  content: "要追加的新内容"
})
```

## 📝 示例代码

### 完整示例：创建技术方案文档
```javascript
async function createTechDoc(title, content) {
  // 第一步：创建空文档
  const createResult = await feishu_doc({
    action: "create",
    title: title
  })
  
  // 第二步：写入内容
  await feishu_doc({
    action: "write",
    doc_token: createResult.document_id,
    content: content
  })
  
  // 第三步：返回结果
  return {
    document_id: createResult.document_id,
    url: createResult.url,
    message: `✅ 文档创建成功：${createResult.url}`
  }
}

// 使用示例
const result = await createTechDoc(
  "自定义平台集成OpenClaw技术方案",
  "# 技术方案\n\n## 目标\n..."
)

console.log(result.message)
```

## 🔍 常见错误

### ❌ 错误1：在create时传入content
```javascript
// ❌ 错误：content参数会被忽略
await feishu_doc({
  action: "create",
  title: "文档标题",
  content: "内容"  // 这个参数无效！
})
```
**结果**：创建了空文档

### ❌ 错误2：忘记第二步写入
```javascript
// ❌ 错误：只创建了空文档，没有写入内容
const result = await feishu_doc({
  action: "create",
  title: "文档标题"
})
// 忘记调用 write 或 append
```
**结果**：文档是空的

## ✅ 检查清单

创建飞书文档时，必须检查：

- [ ] 是否使用了 create + write 两步操作？
- [ ] create 操作是否只传了 title 参数？
- [ ] write/append 操作是否传了 doc_token？
- [ ] write/append 操作是否传了 content？
- [ ] 是否返回了正确的 document_id 和 url？
