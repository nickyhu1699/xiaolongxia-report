# 公众号文章发布工作流

## 📋 完整流程

### 1️⃣ 准备阶段
- 确认文章内容（Markdown格式）
- 确定文章主题和风格
- 规划配图数量和位置（一般3-5张）

### 2️⃣ 生成封面图（AI生图）
**命令：**
```bash
cd ~/.openclaw/skills/ai-image-generator && python3 scripts/main.py "提示词"
```

**提示词模板：**
```
公众号封面图，主题：[文章主题]，[关键词元素]，科技感，渐变背景，现代简约风格，专业感
```

**示例：**
```
公众号封面图，主题：32分钟从想法到产品，扑扑AI开发实录，小龙虾元素，科技感，渐变紫色背景，现代简约风格，专业感
```

**注意事项：**
- AI生图需要1-2分钟
- 生成的图片在：`/Users/sheyujituan/.openclaw/workspace/generated_images/`
- 如果返回目录，取最新的图片文件

### 3️⃣ 生成配图（AI生图）
**配图数量：** 3-5张（根据文章长度）

**配图位置规划：**
- 📸 配图1：痛点场景（在"发现问题"后）
- 📸 配图2：开发过程（在"第一阶段"后）
- 📸 配图3：成果展示（在关键成果后）
- 📸 配图4：商业价值（在数据展示后）

**提示词风格：**
```
小红书风格插画，[场景描述]，扁平化插画风格，温馨配色
小红书风格数据图，[数据描述]，扁平化风格，鲜艳配色
```

### 4️⃣ 创建排好版的HTML
**核心样式规范：**
```html
<section style="padding: 20px;">
  <!-- 普通段落 -->
  <p style="font-size: 16px; line-height: 1.8; margin-bottom: 20px; color: #666;">
    内容
  </p>
  
  <!-- 小标题 -->
  <p style="font-size: 20px; font-weight: bold; margin: 40px 0 15px; color: #333;">
    💡 小标题
  </p>
  
  <!-- 引用块（蓝色） -->
  <p style="padding: 15px; background-color: #e7f3ff; border-left: 4px solid #0066cc; margin-bottom: 15px;">
    引用内容
  </p>
  
  <!-- 引用块（绿色） -->
  <p style="padding: 15px; background-color: #d4edda; border-left: 4px solid #28a745; margin-bottom: 20px;">
    ✅ 成功结果
  </p>
  
  <!-- 引用块（黄色） -->
  <p style="padding: 15px; background-color: #fff3cd; border-left: 4px solid #ffc107; margin-bottom: 20px;">
    🎯 关键信息
  </p>
  
  <!-- 图片占位符 -->
  <!-- IMAGE_X_PLACEHOLDER -->
  
  <!-- 图片插入（替换占位符后） -->
  <p style="text-align: center; margin: 20px 0;">
    <img src="图片URL" style="width: 100%; max-width: 100%; border-radius: 8px;" />
  </p>
  <p style="font-size: 14px; text-align: center; color: #999; margin-bottom: 20px;">
    📸 图片说明
  </p>
</section>
```

**排版要点：**
- ✅ 字体：16px基础，20px标题
- ✅ 行高：1.6-1.8
- ✅ 颜色：#333主文字，#666次要文字，#999辅助文字
- ✅ 间距：段落20px，标题30-40px
- ✅ Emoji装饰：增加趣味性
- ✅ 分段清晰，层次分明

### 5️⃣ 上传图片到微信
**代码：**
```python
from wechat_draft_pusher import WeChatDraftPusher

pusher = WeChatDraftPusher()

# 上传封面图
with open(cover_path, 'rb') as f:
    result = pusher.client.material.add('image', f)
cover_media_id = result['media_id']

# 上传配图
with open(img_path, 'rb') as f:
    result = pusher.client.material.add('image', f)
img_url = result.get('url', '')
```

### 6️⃣ 替换占位符
将HTML中的 `<!-- IMAGE_X_PLACEHOLDER -->` 替换为实际的图片HTML代码。

### 7️⃣ 发布到草稿箱
**代码：**
```python
media_id = pusher.create_draft(
    title="文章标题",
    content=article_html,  # HTML格式
    author='小龙虾🦞',
    digest='文章摘要',
    cover_image=cover_media_id
)
```

### 8️⃣ 清理临时文件
删除AI生成的临时图片文件。

## 📝 注意事项

1. **AI生图时间**
   - 每张图需要1-2分钟
   - 总共5张图约需5-10分钟
   - 避免进程超时

2. **图片处理**
   - AI生图返回路径可能是目录
   - 需要找到最新的图片文件
   - 使用 `max(image_files, key=os.path.getmtime)`

3. **微信API**
   - 封面图使用 `material.add()` 上传永久素材
   - 配图也使用 `material.add()` 获取URL
   - 草稿使用 `create_draft()` 创建

4. **排版风格**
   - 学习 `daily_summary_generator.py` 的风格
   - 保持一致性
   - 适度使用Emoji
