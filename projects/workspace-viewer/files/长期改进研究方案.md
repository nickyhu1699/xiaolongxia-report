## 2️⃣ 加密备份方案研究

### 📋 需求分析

**目标**：加密敏感信息，防止未授权访问

**场景**：
- 邮箱授权码：***HIDDEN***
- 防止记忆文件泄露
- 安全备份

---

### 💡 方案设计

#### 方案A：文件级加密

**使用Python cryptography库**：

```python
from cryptography.fernet import Fernet
from cryptography.hazmat.primitives import hashes
from cryptography.hazmat.primitives.kdf.pbkdf2 import PBKDF2HMAC
import base64
import os

class MemoryEncryption:
    def __init__(self, password):
        # 从密码生成密钥
        salt = b'static_salt'  # 实际使用时应该随机生成并保存
        kdf = PBKDF2HMAC(
            algorithm=hashes.SHA256(),
            length=32,
            salt=salt,
            iterations=100000,
        )
        key = base64.urlsafe_b64encode(kdf.derive(password.encode()))
        self.cipher = Fernet(key)
    
    def encrypt_file(self, input_path, output_path):
        """加密文件"""
        with open(input_path, 'rb') as f:
            data = f.read()
        
        encrypted_data = self.cipher.encrypt(data)
        
        with open(output_path, 'wb') as f:
            f.write(encrypted_data)
    
    def decrypt_file(self, input_path, output_path):
        """解密文件"""
        with open(input_path, 'rb') as f:
            encrypted_data = f.read()
        
        decrypted_data = self.cipher.decrypt(encrypted_data)
        
        with open(output_path, 'wb') as f:
            f.write(decrypted_data)
```

**优点**：
- ✅ 简单易用
- ✅ 加密强度高
- ✅ 支持文件级加密

**缺点**：
- ❌ 需要管理密码
- ❌ 每次访问都要解密

---

#### 方案B：字段级加密

**只加密敏感字段**：

```python
class SensitiveFieldEncryption:
    SENSITIVE_FIELDS = ['授权码：***HIDDEN***
    
    def __init__(self, password):
        self.cipher = MemoryEncryption(password).cipher
    
    def encrypt_sensitive_fields(self, content):
        """加密敏感字段"""
        import re
        
        for field in self.SENSITIVE_FIELDS:
            # 匹配字段值
            pattern = rf'({field}[：:]\s*)([^\n]+)'
            
            def encrypt_match(match):
                prefix = match.group(1)
                value = match.group(2)
                encrypted_value = self.cipher.encrypt(value.encode()).decode()
                return f"{prefix}ENCRYPTED:{encrypted_value}"
            
            content = re.sub(pattern, encrypt_match, content)
        
        return content
    
    def decrypt_sensitive_fields(self, content):
        """解密敏感字段"""
        import re
        
        pattern = r'ENCRYPTED:([^\n]+)'
        
        def decrypt_match(match):
            encrypted_value = match.group(1)
            decrypted_value = self.cipher.decrypt(encrypted_value.encode()).decode()
            return decrypted_value
        
        return re.sub(pattern, decrypt_match, content)
```

**优点**：
- ✅ 只加密敏感部分
- ✅ 其他内容可读
- ✅ 灵活性高

**缺点**：
- ❌ 需要识别敏感字段
- ❌ 可能遗漏敏感信息

---

### 🎯 推荐方案

**混合方案**：
1. **MEMORY.md**：使用字段级加密
2. **备份文件**：使用文件级加密
3. **密钥管理**：使用环境变量或密钥文件

**实现**：
```python

        f.write(encrypted_content)
    
    print(f"✅ 加密备份已创建: {backup_path}")
```
