# 公众号文章排版学习笔记

## 从 daily_summary_generator.py 学到的排版技巧

### 1. HTML样式规范

#### 基础样式
```html
<section style="padding: 20px;">
  <!-- 内容 -->
</section>
```

#### 段落样式
- **普通段落**：
  ```html
  <p style="font-size: 16px; line-height: 1.8; margin-bottom: 20px; color: #666;">
    段落内容
  </p>
  ```

- **小标题**：
  ```html
  <p style="font-size: 20px; font-weight: bold; margin: 30px 0 15px; color: #333;">
    🎯 小标题
  </p>
  ```

- **引用块**：
  ```html
  <p style="padding: 15px; background-color: #f9f9f9; border-left: 4px solid #ff6b6b; margin-bottom: 20px;">
    引用内容
  </p>
  ```

#### 特殊样式
- **强调文字**：使用 `<strong>` 标签
- **emoji装饰**：在小标题前加emoji（🎯📊💡🔥等）
- **居中文字**：
  ```html
  <p style="text-align: center; margin: 30px 0; font-size: 16px; color: #999;">
    <strong>居中内容</strong>
  </p>
  ```

### 2. 图片生成规范

#### 封面图
- **尺寸**：900×500像素
- **生成方式**：使用PIL库
- **内容**：主标题 + 副标题（日期等）
- **颜色**：渐变背景（紫色系）

#### 配图
- **数量**：每个关键段落1-2张图
- **类型**：
  1. 场景图：展示具体工作场景
  2. 数据图：展示效率对比、成果等
- **生成方式**：使用AI图片生成技能
- **命令**：
  ```bash
  cd ~/.openclaw/skills/ai-image-generator && python3 scripts/main.py "提示词"
  ```

#### 图片插入
```html
<p style="text-align: center; margin: 20px 0;">
  <img src="图片URL" style="width: 100%; max-width: 100%; border-radius: 8px;" />
</p>
<p style="font-size: 14px; text-align: center; color: #999; margin-bottom: 20px;">
  📸 图片说明
</p>
```

### 3. 内容结构

#### 开头（吸引人）
- 用emoji开头
- 简短有力的开场白
- 制造悬念或共鸣

#### 正文（分段清晰）
1. **第一段**：背景介绍
2. **第二段**：核心内容（分点叙述）
3. **第三段**：案例/数据支撑
4. **配图穿插**：每2-3段插入1张配图

#### 结尾（互动引导）
- 总结要点
- 引导关注/点赞
- 预告下期内容

### 4. 图片上传流程

1. **生成图片**：使用AI或PIL生成
2. **上传到微信**：
   ```python
   with open(image_path, 'rb') as f:
       result = client.material.add('image', f)
   media_id = result['media_id']
   url = result.get('url', '')
   ```
3. **插入文章**：使用URL插入到HTML中

### 5. 排版技巧总结

✅ **字体**：16px基础，20px标题
✅ **行高**：1.6-1.8，提升可读性
✅ **颜色**：#333主文字，#666次要文字，#999辅助文字
✅ **间距**：段落20px，标题30px
✅ **emoji**：适度使用，增加趣味性
✅ **图片**：居中显示，圆角8px
✅ **引用块**：左边框强调，背景色区分
