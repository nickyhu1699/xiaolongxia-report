## 核心定义

### 一人公司（OPC）

**法律定义**：
- 由一个自然人或法人股东设立的有限责任公司
- 享有独立法人资格与有限责任保护

**AI时代新定义**：
- "碳基智慧（人）+ 硅基执行（AI）"的分工重构
- 个人聚焦战略、创意与决策
- AI承接开发、营销、客服等标准化工作
- 以一人为主导、AI为核心生产力的轻量化组织

**英文术语**：
- OPC: One Person Company
- 也称为：One Person Limited Company

## 关键对比

### 一人公司 vs 个人独资企业

| 对比维度 | 一人有限公司 | 个人独资企业 |
|---------|-------------|------------|
| **法人资格** | ✅ 有 | ❌ 无 |
| **责任承担** | 有限责任 | **无限责任** |
| **投资主体** | 自然人或法人 | 仅限自然人 |
| **税收** | 企业所得税+个人所得税（双重征税） | 仅个人所得税（单层征税） |
| **注册资金** | 约10万元 | 无最低要求 |
| **财产独立性** | 需证明财产独立 | 个人财产与企业财产混合 |
| **风险程度** | 较低 | **较高** |
| **融资能力** | 较强 | 较弱 |

**选择建议**：
- 需要有限责任保护 → 一人有限公司
- 希望税负较轻 → 个人独资企业

## 法律风险

### ⚠️ 核心风险：财产混同

**法律规定**：
- 《公司法》第63条：一人公司股东不能证明公司财产独立于股东自己的财产的，应当对公司债务承担连带责任
- **举证责任倒置**：股东需自证清白

**风险表现**：
1. 个人与公司账户混用
2. 将个人消费计入公司成本
3. 随意调动公司资金
4. 长期借款不还
5. 虚开增值税发票

**后果**：
- 股东对公司债务承担连带责任
- 个人财产被用于偿还公司债务
- 税务稽查和罚款

**5个防范措施**：
1. ✅ 开立独立的公司银行账户
2. ✅ 严格区分个人与公司支出
3. ✅ 每笔支出都有合规发票
4. ✅ 年度财务审计
5. ✅ 规范股东借款程序

## 注册流程

### 必备材料

**1. 公司名称**
- 3-5个备选
- 格式：地域+字号+行业+组织形式
- 建议3-4个字的字号

**2. 注册资金**
- 一人有限公司：约10万元
- 需在五年内实缴（2024年新《公司法》）

**3. 其他材料**
- 法人信息
- 注册地址（必须办公性质）
- 经营范围
- 公司章程

### 注册步骤

**步骤1**：公司核名
- 通过各地政务网"一网通办"入口

**步骤2**：线上提交材料
- 身份证正反面
- 注册地址证明
- 公司章程

**步骤3**：领取营业执照
- 审核通过后领取正副本
- 刻制公章、财务章、合同章、法人章

**步骤4**：银行开户
- 预约银行开户
- 开立基本账户

**步骤5**：税务报到
- 税务局核名
- 主管税务局报道
- 申请发票

**注意事项**：
- 一个自然人只能投资设立一个一人有限责任公司
- 注册地址不能使用住宅地址

## 税务处理

### 税收对比

**一人有限公司**：
1. 企业所得税（25%）
2. 股东分红个人所得税（20%）
- **双重征税**

**个人独资企业**：
1. 仅个人所得税（经营所得）
- **单层征税**

### 5大税务风险

**1. 私户收付款**
- 风险：股东从公司提取资金未归还
- 防范：及时转入对公账户

**2. 财产混同**
- 风险：个人消费与企业支出混淆
- 防范：严格区分，规范记账

**3. 虚开发票**
- 风险：一人公司容易被质疑财产混同
- 防范：业务真实、凭证真实、数据真实

**4. 税务稽查**
- 风险：经营范围过多，数据异常
- 防范：合理设置经营范围

**5. 股东借款**
- 风险：长期借款未还
- 防范：规范程序，及时归还

### "三真"原则

1. 业务真实
2. 凭证真实
3. 数据真实
