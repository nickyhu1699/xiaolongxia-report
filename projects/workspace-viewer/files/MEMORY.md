## 📚 知识库 #知识库

### 一人公司/OPC核心知识 #知识库

**定义与概念**：
- **法律定义**：由一个自然人或法人股东设立的有限责任公司，享有独立法人资格与有限责任保护
- **AI时代新内涵**：在AI协同支持下，个人独立完成产品设计研发至市场投放全链路闭环的新型创业形态
- **核心模式**："碳基智慧（人）+ 硅基执行（AI）"的分工重构

**一人公司 vs 个人独资企业**：

| 维度 | 一人有限公司 | 个人独资企业 |
|------|-------------|------------|
| 法人资格 | ✅ 有 | ❌ 无 |
| 责任承担 | 有限责任（以出资额为限） | **无限责任（个人全部财产）** |
| 投资主体 | 自然人或法人 | 仅限自然人 |
| 税收 | 企业所得税+个人所得税 | 仅个人所得税 |
| 注册资金 | 约10万元 | 无最低要求 |

**注册流程要点**：
1. 公司核名（3-5个备选）
2. 线上提交材料（身份证、地址证明、公司章程）
3. 领取营业执照
4. 银行开户
5. 税务报到

**⚠️ 核心风险 - 财产混同**：
- **法律规定**：《公司法》第63条 - 一人公司股东不能证明公司财产独立于股东自己的财产的，应当对公司债务承担连带责任
- **举证责任倒置**：股东需自证清白，而非债权人举证
- **防范措施**：
  - 开立独立的公司银行账户
  - 严格区分个人与公司支出
  - 每笔支出都有合规发票
  - 年度编制财务会计报告并审计
  - 避免资金随意调动

**OPC的优势**：
1. 有限责任保护（前提：财产独立）
2. 独立法人资格
3. 更易获得融资
4. 永续存在
5. 管理简单
6. 易于转让

**OPC的劣势**：
1. 双重征税（企业所得税+个人所得税）
2. 合规成本高
3. 财产独立举证责任
4. 注册资本要求（10万，五年内实缴）
5. 一个自然人只能设立一个一人公司
6. 不能从事NBFC活动

**AI时代的OPC特点**：
- **组织革命**：有效率的商业组织从5-10人降到1-3人
- **虚拟企业联盟**：OPC之间通过API和智能体协议互联互通
- **核心能力**：不需要成为全能选手，但必须成为某个环节的不可替代者
- **竞争焦点**：从"效率替代"转向"价值创新"
- **典型工具**：OpenClaw、秒哒等AI智能体

**成功案例**（2025-2026）：
1. **自然宸极**：00后清华硕士生，2人团队，对抗极寒装备，瞄准全球市场
2. **梧桐AI**：1人团队，AI+人力资源平台，月均服务超50万人次
3. **深圳政策**：到2027年建成10+ OPC社区，培育1000+ AI创业企业

**"一人独角兽"概念**（OpenAI CEO山姆·奥特曼）：
- "一个人+一万块GPU，将打造数十亿美元的公司"
- 未来可能出现多家真正意义上的"一人独角兽公司"

**OPC成功要素**：
1. 找到自己的优势领域，建立不可替代的价值
2. 熟练使用AI工具，用AI放大个人能力
3. 严格区分个人与公司财产，规范财务管理
4. 建立密集的协作网络，嵌入更大的生态
5. 持续学习，跟上AI技术发展

**适用人群**：

- ✅ 有AI工具使用能力

## 📝 飞书文档操作最佳实践 #流程 #配置

**⚠️ 核心规则：**
1. **feishu_doc的create action只创建空文档** - content参数不会生效
2. **必须用两步法**：先create（创建），再write（写入内容）
3. **或者直接用write** - 如果文档不存在会自动创建

**正确方式**：
```python
# 方式1：两步法（推荐）
feishu_doc(action="create", title="标题")  # 创建空文档
feishu_doc(action="write", doc_token="xxx", content="内容")  # 写入内容

# 方式2：直接write（如果文档不存在会自动创建）
feishu_doc(action="write", title="标题", content="内容")
```

**❌ 错误方式**：
```python
# 错误：create时content参数不会生效
feishu_doc(action="create", title="标题", content="内容")  # 只会创建空文档！
```

**记录时间**：2026-03-20  
**问题发现**：用户反馈飞书文档是空的  
**根本原因**：create action的content参数不生效  
**解决方案**：使用两步法或直接write
