## 🦞 二、OpenClaw是什么？（8分钟）

### 2.1 一句话讲清楚

**OpenClaw = 你的数字员工**

**类比**：
- 传统软件 = 工具（你需要学会用它）
- AI助手 = 员工（你告诉它做什么）

**例子**：
```
传统方式：
你学习Excel → 你做表格 → 你分析数据

OpenClaw方式：
你说"帮我分析这份数据" → AI自动完成 → 给你结果
```

### 2.2 核心特点（3个）

**1. 有记忆**
- 记得你的邮箱、偏好、项目
- 不用每次重复说

**2. 有个性**
- 可以定制性格（比如"小龙虾"）
- 不是冷冰冰的机器人

**3. 有能力**
- 会搜索、会写作、会分析
- 会用各种工具（浏览器、飞书、邮箱...）

### 2.3 与ChatGPT的区别

| ChatGPT | OpenClaw |
|---------|----------|
| 对话工具 | 数字员工 |
| 无记忆 | 有记忆 |
| 通用回答 | 个性化服务 |
| 只能对话 | 能执行任务 |
| 需要科学上网 | 本地运行 |

**关键区别**：ChatGPT是"老师"，OpenClaw是"员工"
