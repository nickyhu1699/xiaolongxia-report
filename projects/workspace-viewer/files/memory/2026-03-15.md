### 7. Phase 7: 上传图片（2-3分钟）
**上传流程：**
```python
from wechat_draft_pusher import WeChatDraftPusher

pusher = WeChatDraftPusher()

# 上传封面图
with open(cover_path, 'rb') as f:
    result = pusher.client.material.add('image', f)
cover_media_id = result['media_id']

# 上传配图
image_urls = {}
for i, img_path in enumerate(image_paths, 1):
    with open(img_path, 'rb') as f:
        result = pusher.client.material.add('image', f)
    image_urls[i] = result.get('url', '')
```

**注意：**
- 使用`material.add`上传永久素材
- 返回`media_id`（封面）和`url`（配图）

### 8. Phase 8: 替换占位符（1分钟）

for i, url in image_urls.items():

</p>"""
    article_html = article_html.replace(placeholder, replace_text)
