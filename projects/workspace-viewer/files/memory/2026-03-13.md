## 🐛 图片路径提取Bug修复

### 问题描述
- **现象**：图片生成了但没有插入文章
- **原因**：路径提取失败
  - 输出：`本地路径：/path/to/image.png`（中文冒号"："）
  - 代码：`cut -d:`（英文冒号":"）
  - 结果：分割失败 → 路径为空 → 图片未插入

### 解决方案
```python
# 修复前
result = os.popen(cmd + ' | grep "本地路径" | cut -d: -f2-').read().strip()

# 修复后
result = os.popen(cmd + ' | grep "本地路径"').read().strip()
if '：' in result:
    result = result.split('：')[1].strip()  # 支持中文冒号
elif ':' in result:
    result = result.split(':')[1].strip()   # 支持英文冒号
```

- **状态**：✅ 已修复
